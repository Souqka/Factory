🔄 Настройка синхронизации с Google Таблицами

📋 Инструкция по настройке:

  1. Создайте новую Google Таблицу
  2. Скопируйте её ID из адресной строки:
    https://docs.google.com/spreadsheets/d/ВОТ_ЭТОТ_ID/edit
  3. Вставьте ID в поле ниже
  4. Откройте: Расширения → Apps Script
  5. Вставьте код скрипта (показан ниже)
  6. Нажмите: Развернуть → Новое развертывание → Веб-приложение
  7. Установите доступ: "Все, включая анонимных пользователей"
  8. Скопируйте URL веб-приложения и вставьте его в поле ниже

📝 Код для Google Apps Script:

Важно: Замените ВАШ_ID_ТАБЛИЦЫ на реальный ID вашей таблицы
